# Multiplatform library template

## What is it?

This repository contains a simple library project, intended to demonstrate a [Kotlin Multiplatform](https://kotlinlang.org/docs/multiplatform.html) library that compiler to JavaScript/TypeScript library and is deployable to [NPM](https://www.npmjs.com/).

The library has only one exported class: `Greeting` that great welcomes your browser.
